Introduction

The service accept regular (long) URLs, store them, and return a shorter link that redirects to the original URL when accessed

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Endpoint to return the specified link match.

Example request:
curl --request GET \
    --get "http://127.0.0.1:8000/api/short/vH5iaL" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://127.0.0.1:8000/api/short/vH5iaL"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Endpoint to create new short link record.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/short" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"url\": \"http:\\/\\/rosenbaum.com\\/eius-aut-eos-et-recusandae.html\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/short"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "url": "http:\/\/rosenbaum.com\/eius-aut-eos-et-recusandae.html"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{

POST api/short

Headers

Content-Type

Accept

Body Parameters

url   string
